Command Line Options

Gems Usage:

java -classpath %CLASSPATH% com.tibco.gems.Gems -p <properties file> -s <servers XML file>

Configuration Files:  
<properties file> Read configuration from the given property configuration file. If no property file is specified Gems will run in view-only mode.
<servers XML file> Read EMS servers configuration from the given XML configuration file. If no file is specified Gems will use servers configuration file speciied in the properties file.
